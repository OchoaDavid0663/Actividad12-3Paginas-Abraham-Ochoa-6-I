# Actividad 12: 3 Páginas en Flutter

## Descripción General

Esta aplicación de Flutter demuestra la navegación básica entre tres pantallas diferentes. La aplicación está estructurada con una página de inicio y dos páginas adicionales, "Página 1" y "Página 2", accesibles a través de botones en la página de inicio.

## Estructura del Proyecto

*   `lib/main.dart`: El punto de entrada de la aplicación, que contiene la configuración de `MaterialApp` y la definición de las rutas de navegación.
*   `lib/home_page.dart`: La pantalla de inicio de la aplicación, que contiene los botones para navegar a las otras dos páginas.
*   `lib/page1.dart`: La primera página adicional.
*   `lib/page2.dart`: La segunda página adicional.

## Plan de Implementación

1.  **Crear el proyecto de Flutter:** Inicializar un nuevo proyecto de Flutter.
2.  **Crear la página de inicio:** Diseñar la página de inicio con dos `ElevatedButton` para la navegación.
3.  **Crear la página 1:** Crear un nuevo archivo para la "Página 1" con un `AppBar` y un `Text` de marcador de posición.
4.  **Crear la página 2:** Crear un nuevo archivo para la "Página 2" con un `AppBar` y un `Text` de marcador de posición.
5.  **Configurar la navegación:** Definir las rutas en `lib/main.dart` para manejar la navegación entre las pantallas.
